# Biometrix

Desktop app with various image processing tools.

## Author

- Oskar Dadan
  
## Technology

C++, Nana, OpenCV

## Key features

- scaling
- change brightness
- convert img to grayscale
- histogram stretching and equalization
- binarization methods and filters
- view histograms
